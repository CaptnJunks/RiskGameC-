README:

Once the game starts, you can choose the amount of players and give them names. In order to make testing easy, by choosing only two players, the distributing armies round would be smaller.

After creating the players, the game will assign countries to the players.

Then, the first ditributing phase begins, in this phase, no players are able to attack.

Once this is done, the setup phase is over, and the game can begin. Following the phases of Reinforcement, Battle, and, if a battle is won, moving territories.

Since there's still no graphic interface, the countries names need to be typed correctly in order for the game to work.

During the reinforcement phases, if you try to reinforce a territory not owned by you, it will simply ignore, and you may choose another one.
